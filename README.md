# 📡 ESP32 + Firebase Realtime Dashboard

Sistema de monitoramento em tempo real baseado em **ESP32 + Firebase**, com visualização via dashboard web desenvolvido para apoio ao projeto de mestrado (Prof. Hilson).

---

## 📌 Visão Geral

Este projeto tem como objetivo integrar dispositivos **ESP32** com sensores de **temperatura e chuva**, enviando dados em tempo real para o **Firebase Realtime Database**.

Uma aplicação web consome esses dados e os exibe em um **dashboard interativo**, separado por equipes, permitindo análise e validação dos experimentos acadêmicos.

---

## 🎯 Objetivo

- Coletar dados via ESP32 com sensores físicos  
- Enviar dados em tempo real para o Firebase  
- Consumir dados em uma aplicação web  
- Exibir informações em um dashboard dinâmico  
- Separar visualização por equipes de pesquisa
